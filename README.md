## ðŸš€ How to Use

1.  **Browse:** Navigate to the folder that matches your current task (e.g., `ux-ui-design`).
2.  **Select:** Open the prompt file (e.g., `error-message-generator.md`).
3.  **Copy & Paste:** Copy the full prompt into your preferred LLM.
4.  **Customize:** **Crucially, replace all bracketed placeholders** (e.g., `[FEATURE_NAME]`, `[USER_GOAL]`) with your project-specific details.
5.  **Analyze:** Review the LLM's output and iterate by providing feedback within the same thread.

> **Tip:** For best results in product engineering, always give the LLM a specific **role** (e.g., "You are a Senior SaaS Product Manager...") and define the **target audience** of the final output
